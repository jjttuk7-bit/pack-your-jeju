# 제주를 담다 PWA 브랜드 아이콘 설계

## 목표

PWA 설치 화면과 스마트폰 홈 화면에서 앱 이름을 `제주를 담다`로 일관되게 표시하고, 기존 영문 `P` 아이콘을 서비스의 감귤 마크로 교체한다.

## 확정 디자인

- 배경은 앱의 기본 표면색과 연결되는 따뜻한 크림색을 사용한다.
- 중앙 심벌은 헤더에서 사용하는 감귤 마크와 같은 형태·색상으로 구성한다.
- 글자는 넣지 않는다. 작은 홈 화면 아이콘에서도 감귤 실루엣이 먼저 인식되게 한다.
- 감귤의 잎과 외곽이 Android 원형·둥근 사각형 마스크에서 잘리지 않도록 마스커블 안전 영역 안에 배치한다.
- 일반 아이콘은 공간을 조금 더 적극적으로 사용하고, 마스커블 아이콘은 더 넓은 안전 여백을 둔다.

## PWA 이름

- Web App Manifest `name`: `제주를 담다`
- Web App Manifest `short_name`: `제주를 담다`
- HTML 문서 제목: `제주를 담다`
- iOS 홈 화면 제목: `제주를 담다`

Manifest 값은 이미 올바르므로 회귀 테스트로 고정하고, 남아 있는 영문 HTML·iOS 메타데이터를 수정한다.

## 아이콘 구성

- Apple touch icon: 180×180 PNG
- PWA 일반 아이콘: 192×192 PNG
- PWA 고해상도 일반 아이콘: 512×512 PNG
- PWA 마스커블 아이콘: 512×512 PNG
- 브라우저 파비콘은 동일한 새 일반 아이콘을 사용한다.

새 파일명을 사용해 기존 `P` 아이콘 URL 캐시와 충돌하지 않도록 한다. PWA의 `id`와 `start_url`은 유지해 앱 정체성과 저장 상태를 바꾸지 않는다.

## 검증

- 생성된 PNG의 실제 픽셀 크기와 RGBA 형식을 검사한다.
- Manifest와 HTML이 새 파일명 및 `제주를 담다` 이름을 참조하는지 테스트한다.
- 프로덕션 빌드 후 생성된 manifest에서 이름, 아이콘 크기, `purpose`를 확인한다.
- 일반 아이콘과 마스커블 아이콘을 시각 검토해 안전 영역과 대비를 확인한다.

## 갱신 안내

새 배포 이후 브라우저와 운영체제가 아이콘을 다시 가져오지만, 이미 설치된 PWA는 운영체제 캐시 때문에 반영이 늦을 수 있다. 즉시 확인이 필요하면 기존 홈 화면 앱을 제거한 뒤 다시 설치한다.
