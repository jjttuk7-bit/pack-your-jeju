# 사업계획서 빨간 표시 문구 삭제 설계

## 목적

사용자가 첨부 이미지에서 빨간색으로 표시한 안내성·평가성 문구만 삭제하고, 최신 기술 중심 사업계획서의 본문·표·색상·10페이지 구성은 유지한다.

## 삭제 범위

1. 표지 부제 `주최 측 공식 목차 대응 · 기술 중심 전면 개정본 · 제품·서비스 개발 부문`
2. `1-1. 개발 서비스의 기능 및 특징(우수성)`의 `(우수성)`
3. `1-2. 공공데이터의 활용 적정성(활용 적정성)`의 `(활용 적정성)`
4. `1-3. 기존 서비스와의 차별성 및 독창성(차별성)`의 `(차별성)`
5. `2-1. 개발 제품 및 서비스를 활용한 창업 계획(사업성)`의 `(사업성)`
6. `2-2. 개발 제품 및 서비스의 사업화 계획(사업화 계획 우수성)`의 `(사업화 계획 우수성)`

## 유지 범위

- 아이템명, 아이템 소개, 제품·서비스, 핵심 기술, 신청 주체, 서비스 URL
- 기술 아키텍처, 공공데이터 활용, 차별성, 개인 창업자 역량, 사업화 계획
- 모든 수치, 표, 색상, 글꼴, 머리말·꼬리말, 10페이지 구성
- DOCX와 PDF의 클릭 가능한 서비스 링크

## 편집 원칙

- 생성기 원본에서 문구를 삭제해 DOCX와 PDF를 재생성한다.
- 표지는 빈 부제를 남기지 않고 제목과 요약표 사이 간격을 자연스럽게 조정한다.
- 제목에서는 괄호 부분만 제거하고 공식 목차 본문은 그대로 둔다.
- 새 설명이나 대체 문구를 추가하지 않는다.

## 검증

- PDF가 정확히 10페이지인지 확인한다.
- 삭제 대상 6개 문구가 DOCX와 PDF 어디에도 없는지 확인한다.
- 기존 필수 기술·수치·개인 창업자 표현과 서비스 링크가 유지되는지 확인한다.
- 10페이지 전체를 이미지로 렌더링해 표지 여백, 제목 줄바꿈, 잘림·겹침을 확인한다.
