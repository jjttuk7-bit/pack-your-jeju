# 경진대회 온라인 접수 입력문안 문서 설계

## 목표

가치더함 온라인 접수 화면의 제목·주요내용을 바로 복사해 입력하고, 대표이미지와 첨부파일을 빠짐없이 준비할 수 있는 실무용 DOCX를 만든다.

## 확정 구성

1. 제목 입력값
2. 주요내용 입력값
3. 온라인 에디터 권장 서식 순서
4. 대표이미지 규격과 권장 내용
5. 첨부파일 선택 안내
6. 저장 전 최종 체크리스트

## 문안 원칙

- 아이템명은 `제주를 담다 (Pack Your Jeju)`로 통일한다.
- 제목은 문제·해결·아이템명이 한 번에 드러나게 작성한다.
- 주요내용은 제안 개요, 문제, 해결, 공공데이터·AI 활용, 차별성, 구현 현황, 사업화, 기대효과 순으로 구성한다.
- 현재 구현과 향후 계획을 구분하고 확인되지 않은 매출·시장규모·팀 경력을 만들지 않는다.
- 웹 에디터에 붙여넣을 때 깨지기 쉬운 복잡한 표는 본문에서 사용하지 않는다.
- 사업계획서의 검증된 수치인 정제 장소 4,422건, 공영주차장 1,557건, 버스정류장 4,271건, 콘텐츠 수정요청 1,686건, 골든셋 12/12를 사용한다.

## 문서 형식

- A4 세로 2~3쪽
- 첫 페이지 상단에 실제 입력할 제목과 주요내용을 복사 영역으로 제공한다.
- 하단 또는 다음 페이지에 대표이미지와 첨부파일 안내를 별도 박스로 표시한다.
- 짙은 제주 녹색과 감귤색을 포인트로 사용하고 본문은 검은색으로 유지한다.

## 검증

- DOCX XML이 정상 파싱된다.
- Word에서 열리고 PDF로 정상 변환된다.
- PDF는 3쪽 이내이며 문단과 체크리스트가 잘리지 않는다.
- 제목·아이템명·서비스 URL·필수 본문 항목이 모두 포함된다.
