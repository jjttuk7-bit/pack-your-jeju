# Landing Journey Infographic Design

## Goal

랜딩 방문자가 제주를 담다를 사용했을 때 여행 준비와 실제 경험이 어떻게 좋아지는지 한눈에 이해하게 한다.

## Story

`발견 → 비교 → 담기 → 경험 기록`의 선형 여정으로 표현한다. 사용법을 반복 설명하지 않고 검색, 근거, 선택, 방문 경험이 하나의 플랜으로 연결되는 제품 가치를 보여준다.

## Layout

- 제주 일러스트 브랜드 소개와 기존 여행 만들기 안내 사이에 배치한다.
- 데스크톱은 네 단계가 하나의 수평 여정선으로 이어진다.
- 모바일은 왼쪽 세로선과 단계가 아래로 이어진다.
- 하단에서 웹 출처, 공공데이터, 실제 경험이 `나의 제주 플랜`으로 합쳐진다.
- 마지막 CTA는 기존 대시보드 진입 동작을 재사용한다.

## Visual Direction

- 짙은 청록 배경에 감귤색 진행점, 흰색 텍스트를 사용한다.
- 독립 카드 나열 대신 구분선과 번호, 아이콘으로 편집형 인포그래픽을 만든다.
- 장식보다 데이터가 연결되는 방향과 단계 차이를 명확히 보여준다.

## Verification

- 데스크톱과 모바일에서 여정선, 문구, CTA가 겹치지 않아야 한다.
- 네 단계가 DOM 순서와 시각 순서에서 동일해야 한다.
- 기존 랜딩 CTA와 대시보드 진입이 유지되어야 한다.
