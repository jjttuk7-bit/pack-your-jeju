# 사업계획서 첫 페이지 핵심 문구 변경 설계

## 목적

첫 페이지 `사업의 중심` 강조 문장을 여행 플랜의 부정형 설명에서 제주 여행을 통한 공공데이터 검증·축적의 긍정형 설명으로 바꾼다.

## 확정 문구

> 제주 여행을 통해 공공데이터는 실제 선택과 방문 속에서 검증되고, 그 결과는 다시 신뢰 가능한 데이터로 축적됩니다.

## 변경 범위

- `scripts/generate_jeju_lifecycle_business_plan.js`의 첫 페이지 강조 문장 한 곳만 수정한다.
- 기존 디자인, 표, 본문, 10페이지 구성과 파일명은 유지한다.
- DOCX와 PDF를 함께 재생성하고 첫 페이지 렌더링으로 잘림과 줄바꿈을 확인한다.

## 검증 기준

- DOCX와 PDF 모두 확정 문구를 포함한다.
- 기존 문구는 남지 않는다.
- PDF는 10페이지를 유지한다.
- 첫 페이지 강조 상자 안에서 문장이 잘리거나 겹치지 않는다.
