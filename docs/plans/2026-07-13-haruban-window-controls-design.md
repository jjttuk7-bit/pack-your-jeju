# Haruban Window Controls Design

## Goal

하루방 에이전트의 모델명을 사용자 화면에서 제거하고, 패널을 본문과 명확히 구분하며 데스크톱에서 이동과 크기 조절이 가능하게 한다.

## Interaction

- 데스크톱에서는 헤더를 드래그해 패널을 이동한다.
- 패널의 네 변과 네 모서리를 드래그해 크기를 조절한다.
- 패널은 화면 밖으로 이동할 수 없고 최소 크기 이하로 줄어들지 않는다.
- 모바일에서는 조작 충돌을 피하기 위해 기존 하단 고정형 패널을 유지한다.

## Visual Direction

- 흰색 패널에 진한 청록색 외곽선과 선명한 그림자를 사용한다.
- 헤더에 이동 가능성을 알려주는 아이콘을 추가한다.
- 모델명 대신 하루방 에이전트가 제공하는 사용자 가치만 설명한다.

## Verification

- TypeScript 검사와 프로덕션 빌드를 통과한다.
- 데스크톱에서 이동, 8방향 크기 조절, 화면 경계 제한을 확인한다.
- 모바일에서 고정형 배치와 입력 영역을 확인한다.
