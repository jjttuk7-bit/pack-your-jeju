# 랜딩 브랜드 스토리 섹션 디자인

## 목표

제주 일러스트레이션과 짧은 소개 문구로 `제주를 담다`의 브랜드 감성을 설명한다.

## 배치

- `오늘의 제주 발견` 사진 갤러리 다음에 배치한다.
- 실사 사진과 기능 설명 사이의 에디토리얼 장면 전환으로 사용한다.
- 데스크톱은 이미지 60%, 소개 40% 구성으로 배치한다.
- 모바일은 이미지 다음에 문구가 이어지는 세로 흐름으로 구성한다.

## 문구

`당신이 만나고 싶은 제주를 담습니다.`를 제목으로 사용한다. 바다, 오름, 시장의 순간을 고르면 나만의 여행이 시작된다는 내용만 짧게 전달한다.
