# Competition Landing Redesign Design

## Goal

랜딩 페이지를 프리 아이펠톤 발표용 설명에서 2026 제주 공공데이터·AI 활용 창업경진대회 제품·서비스 개발 부문에 맞는 실제 서비스 소개 화면으로 전환한다.

## Direction

첫 화면부터 “여행 추천 앱”보다 “제주 공공데이터와 AI가 여행 의사결정을 검증·조립하는 서비스”라는 메시지를 보여준다. 기존 감귤/돌담/제주 톤은 유지하되, 문구와 카드 내용은 경진대회 심사 기준인 공공데이터 활용, AI 기술 활용, 서비스 완성도, 사업화 가능성, 사회적 가치에 맞춘다.

## Scope

- Hero 문구를 경진대회용 제품 포지셔닝으로 변경
- Live Preview 카드를 “우도 미리보기”에서 “날씨·교통·수정요청 기반 의사결정” 사례로 변경
- Trust Engine 섹션을 심사 기준과 데이터 자산 중심으로 재작성
- RAG 섹션을 GPT 강조보다 “공공데이터 → 판정 → AI 조립” 흐름으로 변경
- Trust Cycle/How It Works를 “공모전 제출용 실제 서비스 흐름”으로 정리
- CTA를 대시보드 체험/서비스 시연 진입으로 변경

## Non-goals

- 새 API나 데이터 파이프라인 추가
- 대시보드 기능 변경
- 경진대회 사업계획서 전체 작성

## Success Criteria

- 랜딩에서 아이펠톤/해커톤 중심 문구가 사라진다.
- 첫 화면만 봐도 공공데이터·AI 경진대회 출품작이라는 정체성이 드러난다.
- 현재 이미 구현된 데이터 자산이 심사위원이 이해할 수 있는 말로 보인다.
- `npm run lint`와 `npm run build`가 통과한다.
