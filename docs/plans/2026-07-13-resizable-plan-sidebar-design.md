# Resizable Plan Sidebar Design

## Goal

장소와 방문 피드백이 쌓이는 `MY TRAVEL PLAN` 사이드바를 사용자가 데스크톱에서 좌우로 넓히거나 줄일 수 있게 한다.

## Interaction

- 사이드바 오른쪽 경계를 드래그해 폭을 조절한다.
- 폭 범위는 360~680px이며 오른쪽 후보 영역을 위해 최소 520px을 남긴다.
- 경계를 더블클릭하면 기본 폭 460px로 돌아간다.
- 선택 폭은 로컬 저장소에 보존한다.
- 모바일과 태블릿 단일 열에서는 리사이즈 핸들을 숨긴다.

## Visual

- 기존 두 열 레이아웃과 카드 디자인을 유지한다.
- 열 사이 여백에 얇은 청록색 경계와 작은 세로 손잡이를 표시한다.
- 드래그 중에는 좌우 리사이즈 커서를 페이지 전체에 유지한다.

## Verification

- 최소·최대 폭과 오른쪽 콘텐츠 최소 폭을 확인한다.
- 새로고침 후 폭 복원을 확인한다.
- 모바일에서 단일 열과 텍스트가 깨지지 않는지 확인한다.
