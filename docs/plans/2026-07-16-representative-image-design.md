# 제주를 담다 대표이미지 설계

## 목적

2026 제주 공공데이터·AI 활용 창업경진대회 온라인 접수에 사용할 494×375px 가로형 대표이미지를 제작한다. 한눈에 공공데이터 라이프사이클과 `제주를 담다`의 기술적 차별성을 이해할 수 있어야 한다.

## 확정 디자인

- 형식: PNG, 정확히 494×375px
- 방향: 공공데이터 인포그래픽형
- 색상: 사업계획서와 같은 청록·오프화이트·주황
- 스타일: 심사 서류와 어울리는 단정하고 현대적인 공공데이터 서비스 인포그래픽
- 중앙: 제주도 실루엣과 `제주를 담다`
- 순환 흐름: `공공데이터 → 하루방 RAG → 사용자 입력 → 신뢰 판정 → 실제 여행 → 피드백 → 근거 데이터`
- 상단 문구: `공공데이터·RAG·Trust Engine으로 만드는 제주 여행 신뢰 라이프사이클`
- 하단 문구: `여행으로 검증하고, 데이터로 다시 강화합니다`

## 구성

1. 상단 22%: 핵심 문구를 두 줄 이내로 배치한다.
2. 중앙 60%: 제주 실루엣을 중심으로 7단계 순환 아이콘과 화살표를 배치한다.
3. 하단 18%: 브랜드 메시지를 강조한다.

단계별 아이콘은 데이터베이스, 하루방 대화, 사용자 입력, 방패·체크, 여행 위치, 피드백, 근거 데이터로 구분한다. 작은 크기에서도 의미가 보이도록 선과 면을 단순화한다.

## 제작 방식

- `imagegen`으로 제주·데이터·여행 분위기의 깨끗한 배경 비주얼을 생성한다.
- 한글과 단계명은 생성 이미지에 의존하지 않고 로컬 합성으로 정확히 배치한다.
- 최종 이미지를 494×375px로 리사이즈·크롭하고 PNG로 저장한다.
