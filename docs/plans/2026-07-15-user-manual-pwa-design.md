# 제주를 담다 사용자 매뉴얼 v1.1 설계

## 목표

일반 여행자가 `https://pack-your-jeju.vercel.app/`에 접속해 웹과 PWA 방식으로 서비스를 이용할 수 있도록 최신 12쪽 사용자 매뉴얼 PDF를 만든다.

## 대상과 제외 범위

- 대상: 제주 여행을 준비하는 일반 사용자
- 포함: 서비스 접속, QR, PWA 설치, 여행 조건 입력, 여행팩, 후보 비교, 플랜, 하루방, 피드백, 모바일 문제 해결
- 제외: GitHub, 저장소, 코드, API 키, 개발 환경, 배포, 관리자 화면과 운영자 절차

## 구성

1. 표지 — 서비스명, URL, 클릭 링크, QR 코드, v1.1
2. 빠른 시작 — 6단계와 서비스 주소
3. PWA 설치 — Android·iPhone·PC 설치 방법, 삭제·업데이트·인터넷 연결 안내
4. 첫 화면에서 시작하기
5. 지역·여행 조건·순간 고르기
6. 제주팩 결과 화면 읽기
7. 후보 카드와 신뢰 표시
8. 다른 후보 더 보기
9. 내 여행플랜 만들기·저장·공유
10. 하루방에게 조사 부탁하기
11. 방문 후 피드백
12. 모바일 팁·문제 해결·출발 전 체크

## PWA 안내 원칙

- 배포본에서 `manifest.webmanifest`와 `sw.js`가 200으로 제공되고 `display: standalone`임을 확인한 범위에서 설치 가능하다고 설명한다.
- Android Chrome은 ‘앱 설치’ 또는 ‘홈 화면에 추가’, iPhone Safari는 공유 → 홈 화면에 추가, PC Chrome·Edge는 주소창 설치 아이콘 경로를 안내한다.
- 정적 화면과 앱 셸은 캐시될 수 있으나 최신 검색·공공데이터·날씨·하루방 응답에는 인터넷 연결이 필요하다고 명시한다.
- 설치 버튼의 위치와 문구는 브라우저 버전에 따라 달라질 수 있음을 알린다.

## 시각 설계

- 기존 크림·감귤·민트 색상과 말군고딕을 유지한다.
- 기존 최신 화면 캡처 4장을 재사용한다.
- 표지 QR은 스마트폰으로 스캔 가능한 대비와 여백을 확보한다.
- URL은 텍스트로도 표시하고 클릭 가능한 PDF 링크로 만든다.

## 검증

- PDF 12쪽
- URL 문자열 포함 및 URI 링크 annotation 확인
- QR 코드가 서비스 URL을 인코딩
- `GitHub`, `깃허브`, `repository`, `저장소`, `API 키`, `관리자` 금지어 미포함
- PWA, Android, iPhone, PC와 인터넷 연결 주의 문구 포함
- 전체 페이지 이미지 검토에서 잘림·빈 페이지·과도한 겹침 없음
